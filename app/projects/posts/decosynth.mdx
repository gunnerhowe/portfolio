<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Objective</h2>

DecoSynth was created to provide users with a seamless experience for generating custom art using Stable Diffusion and applying it to wall decor items such as posters and framed canvases. The goal was to allow users to bring their creative ideas to life and have high-quality prints delivered to their doorsteps.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Development Highlights</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Stable Diffusion Integration:</h3>

Implemented AI-powered image generation using Stable Diffusion hosted on Azure, allowing users to create unique artworks based on their prompts.

Managed the Stable Diffusion services and API on Azure for scalable and reliable AI processing.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Customizable Wall Decor:</h3>

Enabled users to preview their generated images on various wall decor items (posters, framed canvases) before making a purchase.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Image Upscaling and Printing:</h3>

Utilized AI-driven upscaling to enhance image resolution for printing.

Integrated printing and shipping workflows to deliver the final products to customers.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>User Authentication and Management:</h3>

Implemented secure user authentication using JWT and NextAuth for session management.

Developed user profiles to save and manage generated artworks.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Payment Processing:</h3>

Integrated Stripe for secure payment processing.

Implemented order management to handle payments and track order statuses.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Email Notifications:</h3>

Set up automated email notifications using Nodemailer to inform users about order confirmations, shipping updates, and promotional offers.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Responsive Design:</h3>

Designed a responsive UI using CSS, SCSS, and Tailwind CSS to ensure a seamless experience across devices.

Utilized React and Next.js for efficient client-side rendering and dynamic content updates.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Third-Party Integrations:</h3>

Integrated Klaviyo for email marketing and customer engagement.

Used AfterShip for managing shipping and logistics.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Objective</h2>
This project aims to create a decentralized application (DApp) that interacts with an ERC-20 smart contract deployed on the Sepolia test network. The goal is to provide users with an interface to check their account balances and demonstrate smart contract integration with a front-end application.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Development Highlights</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>ERC-20 Token Creation:</h3>

Utilized the OpenZeppelin library to create a standard ERC-20 token smart contract named MyToken.

Implemented functions for minting, transferring, and checking balances of tokens.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Truffle Framework:</h3>

Leveraged the Truffle framework for developing, testing, and deploying the smart contract.

Deployed the MyToken contract to the Sepolia test network.


<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Front-End Integration</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Next.js Framework:</h3>

Developed the front-end application using the Next.js framework to create a server-side rendered React application.

Ensured responsive and dynamic UI/UX design for an optimal user experience.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Web3.js Integration:</h3>

Integrated Web3.js to interact with the Ethereum blockchain and the deployed smart contract.

Implemented functions to connect to MetaMask, fetch user accounts, and retrieve token balances.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Alchemy API Utilization:</h3>

Used Alchemy as the blockchain node provider to interact with the Sepolia test network, ensuring reliable and efficient blockchain communication.


<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Features and Functionalities</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>MetaMask Connection:</h3>

Implemented functionality to request account access from MetaMask, allowing users to connect their wallets to the DApp.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Account and Balance Display:</h3>

Displayed connected user accounts and their respective token balances in a user-friendly manner.

Converted balance from wei to ether for readability.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Error Handling and Debugging:</h3>

Included comprehensive error handling to manage issues such as denied account access and failed contract interactions.

Added console logs for effective debugging and monitoring of application state.

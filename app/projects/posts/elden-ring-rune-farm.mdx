<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Objective</h2>
This automation project was designed to streamline the process of rune farming in the game Elden Ring. The goal was to create a bot that can perform repetitive in-game actions efficiently, reducing the manual effort required by players.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Development Highlights</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Win32 API Integration:</h3>

Utilized the Win32 API to interact with the game window, simulate keyboard and mouse events, and control the game efficiently.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Bitmap Recognition with Autopy:</h3>

Implemented bitmap recognition using the Autopy library to locate specific in-game elements on the screen. This includes finding and clicking on assets within the game interface.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Keyboard and Mouse Event Handling:</h3>

Created functions to handle various keyboard and mouse events, enabling the bot to perform complex sequences of actions such as running to specific points, using abilities, and teleporting back.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Automated Game Control:</h3>

Developed a main bot routine that continuously checks for the active game window, executes the farming routine, and handles in-game events to maximize efficiency.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Error Handling and State Management:</h3>

Included checks to ensure the bot is operating within the correct game window, preventing unintended actions if the window focus is lost.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Objective</h2>
This portfolio website was built with the objective of organizing and showcasing my professional experience and personal projects. It serves as a central hub for visitors to learn more about my skills, view my work, and get in touch with me.

<h2 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '22px', fontWeight: '600', textAlign: 'center'}}>Development Highlights</h2>

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>MDX and Markdown Support:</h3>

The website utilizes MDX (Markdown with JSX) to seamlessly integrate React components within markdown files. This enables rich content creation, combining markdown's simplicity with React's flexibility.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Optimized for SEO:</h3>

The website is optimized for search engines with the inclusion of a sitemap, robots.txt, and JSON-LD schema for structured data. These features ensure that the content is easily discoverable and properly indexed by search engines.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>RSS Feed:</h3>

An RSS feed is generated for the blog section, allowing visitors to subscribe and receive updates directly in their preferred RSS reader.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Dynamic OG Images:</h3>

The site dynamically generates Open Graph images for each page, enhancing the visual appeal when links are shared on social media platforms.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Syntax Highlighting:</h3>

Code snippets within blog posts are beautifully rendered with syntax highlighting, making technical content easy to read and understand.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Tailwind CSS:</h3>

Tailwind v4 is used for styling, providing a utility-first approach to CSS. This ensures a consistent design system and rapid UI development.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Vercel Speed Insights / Web Analytics:</h3>

Vercel's Speed Insights and Web Analytics tools are integrated to monitor performance and gather insights on visitor behavior, helping to optimize the user experience.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Geist Font:</h3>

The website uses the elegant and modern Geist font, enhancing readability and aesthetic appeal.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>JavaScript and TypeScript:</h3>

The site is built with both JavaScript and TypeScript, leveraging the strengths of each language. JavaScript provides flexibility, while TypeScript offers static typing for improved code quality and maintainability.

<h3 style={{ color: 'white', fontFamily: 'sans-serif', fontSize: '18px', fontWeight: '600', marginBottom: '-10px'}}>Responsive Design:</h3>

The website is fully responsive, ensuring an optimal viewing experience across a wide range of devices, from desktops to mobile phones.
